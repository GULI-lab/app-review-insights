## 开发规则
- 后端fastapi开发使用uv创建虚拟环境

- 开始任何开发前，先读取 `.Codex/rules/superpower.md`，遵循 Superpowers 技能工作流
- 开发全程调用 Skills 驱动：`/brainstorming` → `/writing-plans` → TDD → code-review → verification → finishing
- 技术约束见 `.Codex/rules/constraints.md`

## MCP 工具

- **Agent 开发**：使用 `docs-langchain` MCP 查询 LangChain 文档，获取 Agent/工具链构建参考
- **UI 设计**：使用 `pencil` MCP 进行网页设计、布局和原型制作
